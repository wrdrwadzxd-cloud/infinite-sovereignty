# Infinite Sovereignty

A decentralized sandbox game that cannot be shut down.

## Core Ideas

- **No central server** → the world stays alive as long as someone is online.
- **9‑layer distributed trust root** → core rules are immutable (not even I can change them).
- **Player data on player's device** → it's actually yours.

## Also Includes

- Multi‑currency & free market
- Family & inheritance system
- Traces of time (buildings decay, land regrows)
- Designed for homebodies (a place to call home)
