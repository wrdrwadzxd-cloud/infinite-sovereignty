# Infinite Sovereignty / 无尽主权

A decentralized virtual world platform that **cannot be shut down**, with **immutable rules** and **true player ownership**.

## 🚀 Core Ideas
- **No central server** – the world lives as long as one player is online.
- **9-layer immutable trust** (Ninefold Mirror) – 3 hardware-anchored cores + 6 libp2p-based layers. Any rule change requires 5/9 consensus.
- **Player sovereignty** – all data stored on player devices; no one can delete or modify your creations.

## 🔧 Technical Highlights
- **libp2p** (P2P networking)
- **CRDTs** (conflict-free state sync)
- **GossipSub** (efficient broadcast)
- **TPM hardware anchoring** (Sybil resistance)
- **DID + social recovery** (identity)
